# NYT React Search
### React-based rendition of the New York Times Article Search application
-------
### Overview
In this app, users are able to search the New York Times API, narrow down your search by entering a keyword and a time frame.  

### Technologies used:

  * Node
 
  * Express
  
  * Handlebars
  
  * MongoDB / Mongoose
  
  * ReactJS
  
  * Bootstrap
